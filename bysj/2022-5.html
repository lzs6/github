Python

  1. python的代码运行方式：
    • 通过新建.py后缀文件，在命令行cd定位到文件目录后输入python 文件名
    • 直接在命令窗口输入python，然后回车，在下一行输入想要运行的代码
  2. python的输入输出：
    • 输入：input()
    • 输出：print()
  3. python语法：

    • 代码缩进

      • 如果省略缩进，Python 会出错
      • 空格数取决于程序员，但至少需要一个
      • 您必须在同一代码块中使用相同数量的空格，否则 Python 会出错

    • 注释

      • 注释以 ＃ 开头，Python 将忽略它们

    • 变量

      • 在 Python 中，变量是在为其赋值时创建的
      • Python 没有声明变量的命令

    • 内置数据类型

    • 获取数据类型

      • 您可以使用 type() 函数获取任何对象的数据类型

    • 必要时使用强制类型转换

      • x = 10 # int
# 把整数转换为浮点数
a = float(x)

    • 字符串

      • 字符串的裁剪：可以使用裁切语法返回一定范围的字符。指定开始索引和结束索引，以冒号分隔，以返回字符串的一部分

        • b = "Hello, World!"
print(b[2:5])

      • 如需获取字符串的长度，请使用 len() 函数

    • 字符串方法：注释：所有字符串方法都返回新值。它们不会更改原始字符串

    • 列表(List数组)：是一种有序和可更改的集合。允许重复的成员。使用[ a,b,c]

    • 元组(Tuple)：是一种有序且不可更改的集合。允许重复的成员。使用( a,b,c)

    • 集合(Set)：是一个无序和无索引的集合。没有重复的成员。使用{ a,b,c}

    • 字典(Dictionary)：是一个无序，可变和有索引的集合。没有重复的成员。使用{ a:1,b:2,c:3}

    • 函数

      • 在 Python 中，使用 def 关键字定义函数

    • 对象

      • 如需创建类，请使用 class 关键字
